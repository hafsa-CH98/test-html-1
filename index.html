les language applicable
  • HTML

    • HTML, ou HyperText Markup Language, est un langage de balisage utilisé pour créer des pages web.
    Il est basé sur le Standard Generalized Markup Language (SGML), un langage de balisage générique qui permet de décrire la structure et le contenu d'un document.
    HTML est un langage textuel simple, composé de balises qui encadrent le contenu d'une page web.
    Les balises sont des mots ou des expressions entourées de symboles et Elles indiquent au navigateur web comment afficher le contenu.

  • CSS

    • CSS, ou Cascading Style Sheets, est un langage de feuille de style utilisé pour contrôler l'apparence des documents HTML et XML.
    Il permet de définir la taille, la couleur, la police, la disposition et d'autres aspects de l'apparence des éléments d'une page web.
    CSS est un langage déclaratif, ce qui signifie qu'il décrit comment les éléments doivent être affichés.
    Les styles CSS sont définis dans des fichiers séparés des documents HTML, ce qui permet de les réutiliser sur plusieurs pages.
